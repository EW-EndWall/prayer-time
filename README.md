# About

Never miss a prayer again with our intuitive prayer time application. Whether you're at home, work, or traveling, stay connected to your faith with accurate prayer times tailored to your location.

# Features

  - Multi-language support (en, tr)
  - Dark mode (dark, light, system mode)
  - Adhan time of 81 provinces
  - 30 days azan time is kept in localstorage

# Quick Installation

## Electron

```bash
$ cd ./electron/
```

### Electron Install

```bash
$ npm install
```
or
```bash
$ yarn install
```
or
```bash
$ pnpm install
```

### Electron Development

```bash
$ npm dev
```
or
```bash
$ yarn dev
```
or
```bash
$ pnpm dev
```

### Electron Build

```bash
# For windows
$ npm build:win

# For macOS
$ npm build:mac

# For Linux
$ npm build:linux
```
or
```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
or
```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
